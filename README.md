# Nicaragua Surf Trip — Juillet

Carte interactive des 30 meilleurs spots de surf au Nicaragua pour juillet, niveau avancé, sud→nord (Rivas → León → Chinandega).

## 🌐 Carte en ligne

👉 [Voir la carte](./nicaragua-surf-spots.html)

## 📁 Fichiers

- `nicaragua-surf-spots.html` — carte Leaflet satellite interactive avec filtres (catégorie / marée / fréquentation / accès)
- `nicaragua-surf-spots.kml` — à importer dans [Google My Maps](https://www.google.com/maps/d/) pour accès mobile & offline

## 🏄 Spots inclus

- **Rivas (sud)** : Yankee, Hermosa SJDS, Gigante, Manzanillo, Colorado, Panga Drops, Santana, Popoyo Main/Outer, Lance's Left, Playgrounds
- **León** : El Tránsito, Asuchillo, El Velero, Shacks, Miramar, Pipes, Outside Miramar, Puerto Sandino (Freight Trains), Salinas Grandes, Las Peñitas/Poneloya
- **Chinandega (nord)** : La Isla Aserradores, Powder Puffs, The Boom, Coconuts, Rocky Point, Meatgrinders, Santa María del Mar, Jiquilillo, Coco Loco

## 📍 Méthode de vérification des coordonnées

- ✓ **17 spots vérifiés** via sources cartographiques indépendantes (latlong.net, mapcarta, worldplaces.me, dateandtime.info, maplandia)
- ~ **8 spots offshore** : offset depuis un repère terrestre vérifié
- ⚠️ **5 spots approximatifs** : coordonnées introuvables dans les bases publiques, position estimée depuis la géographie du littoral (Yankee, Hermosa SJDS, Manzanillo, Santana, Santa María del Mar, Coco Loco)

Toutes les données factuelles (taille de houle, marée idéale, fréquentation, accès, hébergements) sont sourcées sur 2 sources indépendantes minimum. Les spots avec 🚩 n'ont qu'une seule source.
